# 🌐 Priyanka Portfolio

A modern, responsive, and animated personal portfolio website showcasing my skills, projects, and experience as a **Junior .NET Full Stack Developer**.

🔗 my-portfolio-steel-delta-peowo327hk.vercel.app

## 📌 Features

- ✨ Modern UI with gradient and glassmorphism design
- 🎯 Fully responsive (mobile + desktop)
- ⚡ Smooth animations using Framer Motion
- 👩‍💻 Interactive Hero section with typing effect
- 💼 Professional Experience section
- 🛠 Skills categorized in clean layout
- 📂 Project showcase with hover effects
- 📬 Contact section
- 🌙 Dark theme design
